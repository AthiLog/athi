# Jekyll app 설치

`jekyll new [폴더이름]`

`cd [폴더이름]`

```bash
# cmd창에서
bundle exec jekyll serve
```

`localhost:4000` 접속

이 창은 자기 컴퓨터를 기반으로 만든 로컬서버, 즉 자기만 쓸 수 있는 서버, 외부에서 접속할 수 없는 서버이다.

서버를 중지시키고 싶다면 cmd창에서 ctrl + c를 누르면 된다.
